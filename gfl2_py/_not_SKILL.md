# An OCR tool

## What this skill is for

The user wants to convert a visual report into their textual presentation for further processings.

General `report patterns` can be found inside `reference.txt`.

## Workflow

1. **Read `reference.txt`** as the first step.
2. **Ask for a report image to process**
3. **Ask which pattern this report belongs to** 
4. **Translate the report into their textual representation**

## Language path

Python is already installed in this machine. It is preloaded with opencv and tesseract.
Use either or both to fulfill the tas.
